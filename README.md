# CardoAI Client Plugins

Claude Code plugins built by [CardoAI](https://cardoai.com) for use with your Prism data. Install this marketplace once; each plugin below installs separately and can be added or removed independently.

## Setup (one time)

```
/plugin marketplace add CardoAI/cardoai-client-plugins
```

## Plugins

| Plugin | What it does |
|---|---|
| [`prism-analyst`](plugins/prism-analyst/README.md) | Ask questions about your Prism deals in plain language — verified answers with confidence scores and source citations. |

More plugins will be added to this list over time; no change to the setup step above is needed when they land.
